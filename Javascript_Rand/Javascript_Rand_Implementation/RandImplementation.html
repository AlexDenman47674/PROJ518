JavaScript Math.Random Function

Math.floor(Math.random() * 100) + 1) returns a random integer between 1 and 100 (both included). The function has been modified to produce 500 iterations

This script is designed to mirror the random number generator provided by Google as both use the Math.random function

Standard JavaScript has no export method to JSON, so instead the array is converted to JSON format using JSON.stringify in console and then manually copied to the output file
